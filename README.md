## 🚀 대표 프로젝트 (Featured Project)

### 📚 예스24 국내도서 TOP 100 트렌드 추적 및 분석
> **웹 크롤링과 데이터 시각화를 통한 국내 도서 시장 동향 및 소비 트렌드 분석**

- **🔍 프로젝트 정의**: 예스24의 베스트셀러 데이터를 정기적으로 수집하여, 시기별 인기 장르의 변화와 사회적 이슈에 따른 독자 트렌드를 분석한 프로젝트입니다.
- **🛠 사용 기술**: `Python`, `Pandas`, `Matplotlib`, `Seaborn`, `BeautifulSoup`
- **💡 수행 역할 및 핵심 역량**:
  - **데이터 수집/전처리**: BeautifulSoup 기반 웹 크롤러를 제작하여 도서명, 저자, 출판사, 가격, 장르 데이터를 수집하고 정형화했습니다.
  - **트렌드 분석**: 연도별/월별 특정 장르(예: 투자/재테크, 힐링 에세이 등)의 순위 진입 빈도를 분석하여, 당시의 사회·경제적 관심사와의 연관성을 도출했습니다.
  - **데이터 시각화**: `Seaborn`과 `Matplotlib`을 활용해 도서 가격대 분포 및 판매량 추이를 다각도로 시각화하여 데이터 기반의 인사이트를 직관적으로 표현했습니다.
